A Minecraft Plugin that changes a Players name color on their command.  

![screenshot](./images/help.png)

Usage: `/nc color or /nc format-color`  
Example: `/nc bold-aqua`  

|Colors          |Format          |
|----------------|----------------|
|`AQUA`          |`BOLD`          |
|`BLACK`         |`ITALIC`        |
|`BLUE`          |`STRIKETHROUGH` |
|`DARK_AQUA`     |`UNDERLINE`     |
|`DARK_BLUE`     |                |
|`DARK_GRAY`     |                |
|`DARK_GREEN `   |                |
|`DARK_PURPLE`   |                |
|`DARK_RED`      |                |
|`GOLD`          |                |

|`RED`           |                |
|`WHITE`         |                |
|`YELLOW`        |                |
|`RAINBOW`       |                |

